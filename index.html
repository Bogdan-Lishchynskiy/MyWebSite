Web frameworks

React is created and supported by Facebook. Although React is actually not a full-stack framework, it provides only View-level (templating language) and some callback functions to render HTML code, it’s a powerful frontend tool. Visit React

Angular is a javascript open-source framework designed specially for single-page web apps and mobile application development, using MVC architectural pattern. It is not a full-stack, but just a frontend framework dealing basically with your webpages. Visit Angular

Node.js is an open source, cross-platform runtime environment for developing server-side and networking applications This applications are written in JavaScript, and can be run within the Node.js runtime on OS X, Microsoft Windows, and Linux. Node.js also provides a rich library of various JavaScript modules which simplifies the development of web applications using Node.js to a great extent. Visit NodeJS

Rails is a web application framework, written in Ruby, that was derived by David Heinemeier Hansson (DHH) out of work done to create the project management software. Rails aims for flexible, pragmatic, and quick development for web applications. Today, Rails is used on over 1 million websites worldwide and continues to be an attractive full-stack web application platform and a leading open source tool. Visit Rails
